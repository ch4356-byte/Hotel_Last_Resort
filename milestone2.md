## 1. Project Overview

Our project is a website for **Last Resort Hotels**, a management and analytics dashboard designed to help hotel administrators monitor revenue, occupancy, customer satisfaction, and other operational metrics.

The purpose of this site is to display key business information in an interactive, easy-to-understand format that supports better decision-making for hotel management.

## 3. Written Description of Website Content

The website will be a **single-page dashboard** that includes the following sections:

- **Header:** Displays the site logo, name, and navigation links to sections within the page.  
- **Executive Dashboard:** Highlights key performance indicators (KPIs) such as revenue, occupancy rate, and guest satisfaction.  
- **Analytics Section:** Contains charts and tables visualizing financial and operational data over time.  
- **Revenue Section:** Focuses on income trends, monthly reports, and forecasting insights.  
- **Customer Section:** Displays top customers, booking trends, and satisfaction feedback.  
- **Events Section:** Lists and tracks upcoming events, conferences, and group bookings.  
- **Operations Section:** Summarizes internal operations like maintenance tasks, staff activity, and supply tracking.  
- **Footer:** Includes contact details, company links, and copyright information.

## 4. Tools and Technologies

The following tools and technologies will be used to develop the website:

- **HTML5** for webpage structure  
- **CSS3** for styling and layout  
- **JavaScript** for interactivity and dynamic content  
- **Chart.js** for data visualization and chart displays  
- **Responsive design principles** to ensure usability on both desktop and mobile devices
